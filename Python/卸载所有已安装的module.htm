pip 卸载所有已安装的module

要卸载所有已安装的 Python 模块，你可以使用以下方法。但是请注意，这将卸载你系统上的所有 Python 包，包括那些可能是其他应用程序依赖的包。

  1. 使用 pip list 获取所有已安装的包:
bash
pip list --format freeze > installed_packages.txt
  1. 使用 pip uninstall 卸载这些包:
bash
pip uninstall -y -r installed_packages.txt
  1. 删除创建的 installed_packages.txt 文件 (可选):
bash
rm installed_packages.txt

请确保在执行上述操作之前备份你的系统或环境，以防万一。如果你使用 virtual environment (如 venv 或 conda)，那么只会卸载该环境中的包，而不会影响全局安装的包。

使用一条命令呢

使用一条命令卸载所有已安装的Python模块可能有些棘手，但你可以通过将两条命令组合在一起来实现这一目的。以下是一个示例，它首先列出所有已安装的包，然后使用pip uninstall来卸载它们：

bash
pip list --format freeze | grep -v "pip" | xargs pip uninstall -y

这条命令的工作原理如下：

  1. pip list --format freeze：列出所有已安装的包。
  2. grep -v "pip"：从列表中排除pip，这样我们不会尝试卸载它。
  3. xargs pip uninstall -y：对每个列出的包执行pip uninstall命令。

请注意，这条命令将尝试卸载除pip之外的所有包，所以在执行之前，请确保这确实是你想要的操作，并备份任何重要数据。

windows上 cmd 控制台下的命令呢

在Windows的cmd控制台下，你可以使用以下命令来卸载所有已安装的Python模块：

bash
for /f "skip=2 delims=." %i in ('pip list') do pip uninstall -y %i

这个命令的工作方式是：

  1. pip list：列出所有已安装的包。
  2. for /f "skip=2 delims=." %i in ('pip list')：使用for循环迭代pip list的输出，跳过前两行输出（这通常是Package和----），并将每个包的名称赋值给变量%i。
  3. do pip uninstall -y %i：对每个包执行pip uninstall命令。

请注意，在运行此命令之前，确保备份任何重要数据，并确认你确实想要卸载所有Python模块。
