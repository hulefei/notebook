传递Docker环境变量

July 8, 2023
docker 运行如何传递环境变量

Docker 提供了多种方式来传递环境变量给您的容器。以下是两种常见的方式：

1. Docker命令行参数 -e 或 --env

在运行Docker容器时，可以使用 -e 或 --env 参数传递环境变量。例如：

bash
docker run -e MY_ENV_VAR=my_value my_image

这里，MY_ENV_VAR 是环境变量的名称，my_value 是其值。您可以传递任意数量的环境变量，只需要为每个环境变量添加 -e 或 --env。

2. Dockerfile 中的 ENV 指令

在 Dockerfile 中，可以使用 ENV 指令设置环境变量。例如：

Dockerfile
FROM my_base_image
ENV MY_ENV_VAR=my_value

在这里，MY_ENV_VAR 是环境变量的名称，my_value 是其值。当使用此 Dockerfile 构建镜像并运行容器时，这些环境变量将自动设置。

还有一个选项是使用 .env 文件或 --env-file 参数传递一个包含环境变量的文件。例如：

bash
docker run --env-file ./env.list my_image

在这里，env.list 文件包含了一个或多个环境变量和其值。

以上是 Docker 提供的传递环境变量的主要方式，您可以根据需要选择最适合您的方式。
