Linux Screen 使用指南

July 7, 2023
linux screen 使用

screen 是一个极其有用的命令行工具，它可以帮助你在单个终端窗口中管理多个会话。如果你在长时间运行的程序中失去了网络连接，或者需要在多个任务之间进行快速切换，那么 screen 就会派上用场。

以下是一些基本的 screen 命令：

  • 启动 screen 会话：

    bash
    screen

  • 启动一个有名称的 screen 会话：

    bash
    screen -S mysession

  • 列出所有 screen 会话：

    bash
    screen -ls

  • 重新连接到一个存在的 screen 会话：

    bash
    screen -r

    或者如果有多个会话，你需要指定会话名或者会话 ID：

    bash
    screen -r [session-name or session-ID]

  • 在 screen 会话内部，你可以使用 Ctrl-a 和其他键的组合来进行不同的操作，例如：

    • Ctrl-a d：从当前会话中断开（detach），但保持它在后台运行。

    • Ctrl-a c：创建一个新的 screen 窗口。

    • Ctrl-a n 和 Ctrl-a p：切换到下一个或上一个 screen 窗口。

    • Ctrl-a [0-9]：切换到指定编号的 screen 窗口。

    • Ctrl-a k：关闭当前窗口。

  • 结束 screen 会话：只需在所有窗口中退出 shell 即可。例如，你可以在每个窗口中键入 exit，或者使用 Ctrl-d 组合键。

请注意，如果你用的是 tmux 或者 Byobu，那么这些命令可能会有所不同。具体请参考相应的手册页（man pages）。
