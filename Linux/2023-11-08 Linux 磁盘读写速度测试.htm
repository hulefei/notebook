linux 目录读写测速

在 Linux 中测量磁盘的读写速度有多种方法，下面列举了一些常用的工具和命令。

  1. dd 命令 dd 命令是用于测量磁盘写速度的基本命令。要测试读速度，通常需要首先创建一个测试文件，然后读取该文件来测量速度。例如：

    写速度：

    bash
    dd if=/dev/zero of=~/testfile bs=1G count=1 oflag=dsync

    这个命令会创建一个 1 GB 的文件并测量创建所需的时间。

    读速度：

    bash
    dd if=~/testfile of=/dev/null bs=1G

    这个命令会读取之前创建的文件并测量所需时间。

  2. hdparm 命令 hdparm 是专门用于测试磁盘读取速度的工具。可以使用如下命令：

    bash
    hdparm -Tt /dev/sda

    这个命令会对 /dev/sda 这个设备执行缓存读取测试和磁盘读取测试。

  3. iotop 或 iostat 这些工具用于监控系统的 I/O 操作，并实时显示读写速度。

    安装 iotop:

    bash
    sudo apt-get install iotop

    运行 iotop:

    bash
    sudo iotop

    安装 iostat:

    bash
    sudo apt-get install sysstat

    运行 iostat:

    bash
    iostat -mx 2

    以上命令会每2秒输出一次磁盘的读写速度。

  4. fio 工具 fio 是一个功能强大的磁盘 I/O 性能测试工具。它支持多种类型的测试，并可以详细地配置测试的各个参数。

    安装 fio:

    bash
    sudo apt-get install fio

    使用 fio 进行测试：

    bash
    fio --name=Test --eta-newline=5s --filename=fio-tempfile --rw=read --size=500M --io_size=1G --blocksize=64k --ioengine=libaio --direct=1 --numjobs=1 --runtime=60 --group_reporting

  5. bonnie++ 工具 bonnie++ 是另一个用于测试文件系统性能的程序。

    安装 bonnie++:

    bash
    sudo apt-get install bonnie++

    使用 bonnie++ 进行测试：

    bash
    bonnie++ -d /mnt -s 4G -n 0 -m TEST -f -b -u root

请注意，上述命令可能需要根据你的系统环境或者具体需求进行适当修改。在运行任何磁盘性能测试时，请确保备份重要数据，因为某些测试可能会覆盖目标文件或磁盘上的数据。

查看文件大小

在 Linux 中，查看文件大小的最常用命令是 ls 和 du。

  1. 使用 ls 命令

    如果你只是想快速查看一个文件的大小，可以使用 -lh 选项，这会以易读的格式（例如 KB、MB、GB）显示文件大小：

    bash
    ls -lh filename

    这里的 filename 是你要查看大小的文件。-l 选项表示长格式列出信息，-h 表示“人类可读”的格式。

  2. 使用 du 命令

    如果你想查看文件或目录占用的磁盘空间量，可以使用 du 命令：

    bash
    du -sh filename

    在这里，-s 表示汇总每个参数的总用量，而 -h 同样表示以人类可读的格式显示（如 KB、MB、GB）。

  3. 查看目录下所有文件的大小

    如果你想查看一个目录下所有文件的大小，可以组合 ls 和 sort 命令进行排序：

    bash
    ls -lhS | less

    -S 选项会按文件大小排序。通过管道使用 less 可以方便地浏览输出结果。

  4. 使用 find 命令查找特定大小的文件

    如果你需要查找特定大小的文件，可以使用 find 命令：

    bash
    find /path/to/search -type f -size +10M

    上面的命令会查找在 /path/to/search 路径下，大小超过 10MB 的文件。

  5. 使用 stat 命令查看文件详细信息

    要获取关于文件的详细信息，包括精确大小，可以使用 stat：

    bash
    stat filename

    这将显示有关文件的详细信息，包括大小、inode 信息、修改时间等。

  6. 使用 ncdu

    如果你想通过一个交互式的界面来查看文件和目录的大小，可以使用 ncdu 命令：

    bash
    ncdu /path/to/directory

    ncdu 需要安装才能使用。它可以提供一个类似图形界面的方式来浏览不同文件和目录的大小。

    安装 ncdu:

    bash
    sudo apt-get install ncdu

    或者在其他基于 RPM 的发行版上：

    bash
    sudo yum install ncdu

    或者使用 dnf:

    bash
    sudo dnf install ncdu

选择适合你需求的命令来查看文件大小。记住，大多数命令在处理非常大的目录时可能会需要一些时间来计算所有的文件大小。
