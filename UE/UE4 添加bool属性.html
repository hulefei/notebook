UE4 添加bool属性

July 6, 2023
UE4 如何在属性面板中添加bool属性，并添加Onchange回调

在Unreal Engine 4中，你可以在C++类中定义一个bool属性，并使用UPROPERTY宏进行注解，以便它在编辑器的属性面板中显示。你还可以为这个属性添加一个OnChange回调，使你能够执行一些逻辑，当属性发生改变时。

以下是一个简单的例子：

首先，你需要在你的类头文件中添加bool属性以及OnPropertyChange的函数：

cpp
#pragma once

#include "CoreMinimal.h"
#include "GameFramework/Actor.h"
#include "MyActor.generated.h"

UCLASS()
class MYPROJECT_API AMyActor : public AActor
{
	GENERATED_BODY()

public:	
	AMyActor();

protected:
	virtual void BeginPlay() override;

public:	
	virtual void Tick(float DeltaTime) override;

	UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "My Category")
	bool bMyBoolProperty;

	UFUNCTION()
	void OnPropertyChange();

#if WITH_EDITOR
	virtual void PostEditChangeProperty(FPropertyChangedEvent& PropertyChangedEvent) override;
#endif
};

然后在源文件中添加对应的函数实现：

cpp
#include "MyActor.h"

AMyActor::AMyActor()
{
	PrimaryActorTick.bCanEverTick = true;
	bMyBoolProperty = false;
}

void AMyActor::BeginPlay()
{
	Super::BeginPlay();
}

void AMyActor::Tick(float DeltaTime)
{
	Super::Tick(DeltaTime);
}

void AMyActor::OnPropertyChange()
{
	if (bMyBoolProperty)
	{
		UE_LOG(LogTemp, Warning, TEXT("bMyBoolProperty is true"));
	}
	else
	{
		UE_LOG(LogTemp, Warning, TEXT("bMyBoolProperty is false"));
	}
}

#if WITH_EDITOR
void AMyActor::PostEditChangeProperty(FPropertyChangedEvent& PropertyChangedEvent)
{
	Super::PostEditChangeProperty(PropertyChangedEvent);

	if (PropertyChangedEvent.GetPropertyName() == GET_MEMBER_NAME_CHECKED(AMyActor, bMyBoolProperty))
	{
		OnPropertyChange();
	}
}
#endif

在这个例子中，我们首先定义了一个名为bMyBoolProperty的bool属性，并且使用UPROPERTY宏进行注解，以便它可以在编辑器中被编辑。

然后，我们在AMyActor类中定义了一个OnPropertyChange函数，这个函数会在bMyBoolProperty属性发生改变时被调用。

最后，我们覆盖了PostEditChangeProperty函数，这个函数会在任何属性被编辑器改变时被调用。我们在其中检查了改变的属性是否是bMyBoolProperty，如果是，就调用OnPropertyChange函数。

需要注意的是，PostEditChangeProperty函数在运行时并不会被调用，它只会在编辑器中修改属性值时被调用。如果你需要在运行时对属性的改变做出反应，你可能需要在Tick函数或者其他地方进行检查。

此代码是在C++中完成的。如果你是在蓝图中完成的，可以利用蓝图的事件和属性系统来实现类似的功能。
